Setup:
1. Create Supabase free project.
2. Open SQL Editor.
3. Paste and run supabase_products_table.sql.
4. Go to Supabase Project Settings -> API.
5. Copy Project URL and anon public key.
6. Admin panel -> Supabase Settings -> paste both -> Save -> Test Supabase.
7. Upload products.

Important:
For all visitors to see products without entering settings in browser, paste Supabase URL and anon key into assets/config.js, then upload to GitHub.
